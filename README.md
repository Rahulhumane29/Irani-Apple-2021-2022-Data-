🍎 Irani Apple Export Data Dashboard | Power BI Project

This project presents an interactive Power BI dashboard analyzing Iran’s apple export data, providing insights into trade patterns, export destinations, and overall performance.

📊 Data Source
Data was collected from the World Bank WITS (World Integrated Trade Solution) platform.
The dataset is originally sourced from the UN Comtrade database, which compiles official trade statistics reported by Iran’s government agencies (e.g., customs authorities).
The analysis uses publicly available data for 2021–2022, as more recent data (2023–2025) is either restricted or not fully published.
🎯 Project Objectives
Analyze export trends of apples from Iran
Identify top importing countries
Visualize year-wise export performance
Provide actionable insights using data visualization
🛠️ Tools & Technologies
Power BI – Dashboard creation & visualization
Excel / CSV – Data preprocessing
Data Cleaning & Transformation – Power Query
📈 Key Features
Interactive and dynamic dashboard
Country-wise export analysis
Year-wise comparison (2021 vs 2022)
Clean and user-friendly visual design
📌 Insights
Highlights major export markets for Iranian apples
Shows variation in export volume/value across years
Helps understand global trade positioning
